About me

Hello! My name is Ngonidzashe Fungai Zvenyika, a 19-year-old studying Computer Science at Stellenbosch University. My interest in computing began in high school, where I explored Computer Science, Mathematics, and Physics, sparking my passion for logical problem-solving. In 2023, I immersed myself in an online web development course, expanding my skills and creating a portfolio to showcase my abilities. Beyond academics, I enjoy nature, exercise, and literature.

My work

CV-Application

This application served as a conduit for me to implement React and utilize Vite as a build tool. Through this endeavour, I've effectively streamlined and customized the process of generating my own CVs.

Battleship

The primary objective driving the development of this Battleship game was to adhere to the principles of test-driven development. Employing Jest as the testing framework, I ensured that each function pivotal to the application's logic underwent comprehensive unit testing.

Weather Application

I built this project in order to learn about API integration in web applications. To achieve this, I used asynchronous JavaScript functions for accurate data retrieval. Users may switch from the default location using the search bar functionality.

To-Do List Application

To create a robust to-do list application, I had to make use of the solid design principles. I utilized JSON and the Web Storage API for data persistence, Webpack for bundling, and npm for package management. Additionally, I leveraged packages like date-fns to streamline date formatting.

Tic Tac Toe

The backbone of this tic-tac-toe game is a recursive minimax algorithm which drives the unbeatable AI opponent. Throughout the development process, I learnt about key concepts including recursion, the module pattern, and factory functions.

Binary Search Tree

Developing my own binary search tree enabled me to comprehend the relationships between the nodes and explore the various traversal methods. This project aligns with others I've completed, including sorting algorithms and various data structures. You can explore these in detail by accessing their repositories on my GitHub.
