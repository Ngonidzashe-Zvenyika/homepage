My work

CV-Application

This application was a conduit for me to implement React and to use vite as a build tool. As a result I have streamlined and customised the process of generating my own cv's.

Battleship

The purpose behind this project was to make use of test-driven development principles. I used Jest as my testing framework and ensured that each function with a significant role in the application's logic had a unit test.

Weather Application

The goal behind building a weather application was to tackle api usage in web applications. In line with this, I needed to use asynchronous javascript functions to ensure that data was retrieved correctly. One may use the searchbar to change from the default location.

To-Do List Application

In order to develop a to-do list application, I had to keep the solid design principles in mind. I made use of json and the web-storage api for application persistence, webpack for bundling and npm for installing packages like date-fns to format information.

Tic Tac Toe

The backbone of this tic tac toe game is a recursive minimax algorithm that serves as the unbeatable ai opponent. Amongst my learnings, were conceptes such as recursion, the module pattern and factory functions.

Binary Search Tree

Implementing my own binary search tree allowed me to grasp how each of the nodes are related and understand the various ways in which they can be traversed. In line with this, I have completed similar projects for sorting algorithms and other data structures. These are public respositories that may be viewed on my github.
